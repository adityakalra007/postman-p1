    Try Running it on Postman 
    
    
Open Postman.

Create a new request tab.

Set the request type to POST.

In the URL bar, enter:

http://localhost:5000/contact

Go to the Body tab.

Select raw and choose JSON from the dropdown.

Enter this sample JSON:
